| **Object Name** | **Description**                                                               |
|-----------------|--------------------------------------------------------------------------------|
| **User**        | User accounts                                                                  |
| **Business**    | Businesses / Organizations                                                     |
| **Service**     | Services offered by each business                                              |
| **Appointment** | Service appointments that can be scheduled between users and businesses        |
